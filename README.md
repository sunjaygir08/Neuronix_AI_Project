AI Chatbot Web Application

## Overview
This project is a simple AI Chatbot web application built using Flask. The chatbot takes user input from a browser, processes it, and returns a generated response. It provides a clean and interactive interface where users can chat with the bot in real time.

## Features
- Interactive chat interface
- Real-time chatbot responses
- Simple and clean UI
- Easy to deploy on local machine
- Flask backend server

## Tech Stack
| Component | Technology Used |
|----------|----------------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python (Flask) |
| Logic / Processing | Python |
| Server | Localhost (Flask Development Server) |

## How to Run the Project

* Step 1 — Install Python
Make sure Python is installed on your system.  
Check version: using commands in terminal.
Command: python --version

* Step 2 — Install Required Modules
Navigate to the project folder:
Command: cd Ai Chatbot/
Install Flask:
Command: pip install flask

* Step 3 — Run the Application
Command: python app.py

* Step 4 — Open in Browser
Once the server starts, "Ctrl+ click" to open the link in your browser:
http://127.0.0.1:8000/
You should now see the chatbot interface in browser.
